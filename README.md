# Hyper Sync

Módulo de sincronização de dados para HyperShop. Integra Mercado Livre e Tiny ERP para consolidar dados de vendas, anúncios e métricas em um banco relacional.

## Setup

1. Crie um ambiente virtual: `python -m venv venv`
2. Ative: `.\venv\Scripts\activate`
3. Instale dependências: `pip install -r requirements.txt`
4. Configure `.env` usando `.env.example` como base.
5. Execute: `python main.py`
